# serial-port-interface

This project is a Serial Port Terminal application built with Python and Tkinter. It provides a
user-friendly graphical interface to communicate with serial devices, making it suitable for
debugging, testing UART/RS422 simulations, and hardware communication.
Key Features
● Dynamic Port Selection: Automatically detects available COM ports on the system.
● Flexible Configuration: Support for various Baud Rates, Parity, Data Bits, and Stop Bits.
● Bidirectional Communication: Features both data transmission (TX) and real-time data
reception (RX) via a scrolled text area.
● Status Tracking: Real-time connection status updates.
Technical Details
The core logic is contained in karsirealtermsu.py, which utilizes the pyserial library for robust
serial communication.
Requirements:

● Python 3.x
● pyserial library (Install via: pip install pyserial)


Bu proje, Python ve Tkinter kullanılarak geliştirilmiş bir Seri Port Terminal uygulamasıdır. Seri
cihazlarla iletişim kurmak için kullanıcı dostu bir grafik arayüz sunar; UART/RS422
simülasyonları, donanım testleri ve cihaz haberleşmesi için idealdir.
Temel Özellikler
● Dinamik Port Seçimi: Sistemdeki mevcut COM portlarını otomatik olarak algılar.
● Esnek Yapılandırma: Farklı Baud Rate, Parity, Data Bits ve Stop Bits ayarlarını destekler.
● Çift Yönlü İletişim: Hem veri gönderimi (TX) hem de kaydırılabilir metin alanı üzerinden
gerçek zamanlı veri alımı (RX) sağlar.
● Durum Takibi: Bağlantı durumunu gerçek zamanlı olarak takip eder.

Teknik Detaylar
Uygulamanın ana mantığı karsirealtermsu.py dosyasında yer almaktadır ve kararlı bir seri
iletişim için pyserial kütüphanesini kullanır.
Gereksinimler:

● Python 3.x
● pyserial kütüphanesi (Yüklemek için: pip install pyserial)
